# FLOOD ALERTING AND RESCUE SYSTEM

## Overview
This project is designed to detect rising water levels and provide emergency alerts using Arduino Uno and GSM technology. The system helps in early flood warning and improves emergency response.

## Features
- Real-time water level monitoring
- Emergency SMS alert system
- Buzzer and LED warning indications
- Low-cost embedded system design

## Components Used
- Arduino Uno
- Water Level Sensors
- GSM Module
- Buzzer
- LEDs
- Power Supply Circuit

## Technologies Used
- Arduino IDE
- Embedded C / Arduino Programming
- GSM Communication
- Sensor Interfacing

## Working Principle
The water level sensors continuously monitor water levels. When the water exceeds a predefined threshold, the Arduino activates the buzzer and LEDs and sends alert messages using the GSM module.

## Applications
- Flood monitoring systems
- Disaster management
- River and dam monitoring
- Public safety alert systems
